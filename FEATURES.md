# AutoLaunch Features

## Launches (Calendar)
- Schedule posts across multiple social platforms
- Calendar view to see all scheduled content at a glance
- Create, edit, and delete posts
- Post status tracking: scheduled, published, failed

## Analytics
- Track impressions, engagements, and follower growth
- Visual bar chart for performance over time
- Per-platform breakdown (followers, engagement rate, posts)
- Date range filters: 7 days, 30 days, 90 days

## Media Library
- Upload and store images and videos
- Grid and list view toggle
- Filter by file type (all, image, video)
- File metadata: name, size, date

## AI Agents
- Pre-built agents: Content Writer, Hashtag Optimizer, Engagement Analyzer, Trend Spotter
- Run agents to automate content tasks
- Built-in AI chat panel for on-demand assistance

## Plugs (Plugins)
- Extend platform functionality with plugins
- Enable or disable plugins per workspace

## Integrations
- Connect social accounts: X (Twitter), LinkedIn, Instagram, Facebook, YouTube, Reddit, Discord, Telegram
- View connection status per platform
- One-click connect and disconnect

## Settings
- Profile management: name, email, timezone
- Notification preferences
- Team member management with roles
- API key generation and management
- Danger zone: account deletion

## Billing
- Free, Pro, and Business pricing plans
- Plan comparison and upgrade flow
- Subscription status display

## Auth
- Login and registration with mock auth
- Cookie-based session management
- Auto-redirect on unauthorized access

## General
- Dark mode by default, light mode toggle
- Postiz-inspired design system with CSS variables
- Fully responsive layout
- Next.js 16 App Router with TypeScript and SCSS Modules
